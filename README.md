# Social Connect 

 ## 📝Sobre o Social Connect:

O *Social Connect* nasceu da necessidade de criar sistemas mais abertos e conectados. Em um cenário onde a fadiga de senhas afasta usuários, este projeto demonstra como utilizar HTML, CSS e tecnologias de ponta para:

* *Autenticar* via provedores externos com segurança.
* *Sincronizar* dados para criar perfis dinâmicos.
* *Fomentar* o engajamento através de grafos sociais.
* *Garantir* a privacidade dos dados seguindo boas práticas de desenvolvimento.
